*[ddPCR]: digital droplet PCR — the partitioned PCR assay vflank's flanks feed into
*[MAF]: Mutation Annotation Format — TSV of called variants (1-based, fully closed)
*[VCF]: Variant Call Format — tab-delimited variant file (1-based positions)
*[BND]: Breakend — a generic VCF SV record describing one side of a junction
*[CT]: Connection Type — Delly's strand-orientation tag for an SV (e.g. 3to5)
*[SV]: Structural Variant — a large rearrangement (deletion, duplication, inversion, translocation, fusion)
*[TRA]: Translocation — an inter-chromosomal structural variant
*[DEL]: Deletion
*[DUP]: Duplication
*[INV]: Inversion
*[INS]: Insertion
*[SNP]: Single-Nucleotide Polymorphism — a common single-base population variant
*[indel]: insertion or deletion
*[het]: heterozygous — two different alleles at a locus
*[hom]: homozygous — two identical alleles at a locus
*[AF]: Allele Frequency — fraction of chromosomes carrying the allele (AC / AN)
*[AC]: Allele Count — number of chromosomes carrying the allele
*[AN]: Allele Number — total number of called chromosomes
*[gnomAD]: Genome Aggregation Database — the population-frequency reference
*[REF]: reference allele
*[ALT]: alternate (variant) allele
*[BAM]: Binary Alignment Map — compressed aligned-read file
*[CIGAR]: Compact Idiosyncratic Gapped Alignment Report — per-read alignment string (M/I/D/N…)
*[MQ]: Mapping Quality — Phred-scaled confidence a read is correctly placed
*[BQ]: Base Quality — Phred-scaled confidence in a single base call
*[IUPAC]: International Union of Pure and Applied Chemistry — degenerate base codes (R, Y, S, W, K, M…)
*[FASTA]: text format for nucleotide sequences (`>header` + sequence lines)
*[hg19]: human genome build 19 (== GRCh37)
*[GRCh37]: Genome Reference Consortium human build 37 (== hg19)
*[hg38]: human genome build 38 (== GRCh38)

*[API]: Application Programming Interface
*[OIDC]: OpenID Connect — the token flow used for PyPI Trusted Publishing

*[CI]: Continuous Integration
